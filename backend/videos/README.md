# Videos Directory

Place your video files in this directory. Supported formats:
- .mp4
- .webm
- .ogg
- .mov
- .avi

The videos will be automatically detected and made available on the public monitoring page.

The video player will:
- Auto-play the first video on page load
- Loop through all videos
- Allow users to select specific videos
- Provide standard video controls (play, pause, seek, volume)
